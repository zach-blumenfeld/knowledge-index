Knowledge Index (ki)

The safe, easy, fast search engine for your knowledge base.

$ curl -sSfL https://knowledge-index.ai/install.sh | bash

Prefer Python tooling? uv tool install knowledge-index

Built for agents and humans

Wire it into any AI coding agent, or query straight from the CLI. Understand → search → retrieve, with no prior graph expertise required.

Vendor-neutral, instant to set up

No LLM work or embeddings at index time. The index is disposable cache you can rebuild with one ki index call — free, fast, and yours.

A search index, not a store

Point ki at a folder of markdown — notes, docs, a wiki — and it syncs to a Neo4j knowledge-graph index you can search and navigate in seconds. Your source files are never modified.
